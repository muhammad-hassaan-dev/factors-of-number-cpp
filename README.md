# Factors of a Number (C++)

This is a basic C++ program that prints all factors of a given number using a for loop.

## 💡 Concept Used:
- for loop
- if condition
- modulus operator (%)
- user input (cin)
- output (cout)

## 📌 How it works:
1. User enters a number (n)
2. Loop runs from 1 to n
3. Each number (i) is checked:
   - If n % i == 0 → it is a factor
4. Factor is printed

## ▶️ Example:
Input:
6

Output:
1 2 3 6

## 🚀 Purpose:
This program helps beginners understand loops, conditions, and factor logic in C++.
